# create-ppv-app

CLI to quickly create React application with zero config

Project will be initialized using `Vite`.

`Typescrict`, `Tailwind`, `ESLint`, `Prettier` will be included with needed configurations

## Command to run

### npm
```
npx create-ppv-app
```
### yarn
```
yarn create ppv-app
```
### pnpm
```
pnpm dlx ppv-app
```
